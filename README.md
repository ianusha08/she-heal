# 🌸 SheHeal

A women's health companion focused on menstrual, hormonal, and reproductive wellness. SheHeal aims to simplify health tracking by bringing cycle monitoring, symptom logging, wellness habits, and personalized insights into one platform.

> 🚧 **Status:** Currently in development.

## ✨ Planned Features

- 🌸 Menstrual cycle tracking
- 📊 Symptom & hormonal health monitoring
- 💧 Wellness tracking (water, sleep, weight & activity)
- 💊 Medication and appointment reminders
- 📈 Health insights & progress dashboard
- 🤖 AI-powered wellness assistant

## 🛠️ Planned Tech Stack

- React
- TypeScript
- Tailwind CSS
- Node.js & Express
- MongoDB
- Firebase Authentication
- Vercel
